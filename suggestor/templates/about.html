Spotify

This project generates a list of song recommendations based on a song. The dataset we chose is a collection of 160,000+ tracks on Spotify, found on Kaggle.

Input

Our input is a song selected from a list. Our model analyzes the song based on these categories:

acousticness, danceability, energy, explicitness

instrumentalness, liveliness, key, loudness

mode, popularity, speechiness, tempo, valence

Output

After a model analysis, the output of the model is a list of recommendations that are similar to the input song that a listener might like.
